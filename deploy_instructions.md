# Deploy Instructions for DailyTen Website

## Step 1: Create GitHub Repository
1. Go to https://github.com
2. Create free account (no credit card needed)
3. Click "+" → "New repository"
4. Repository name: dailyten-website
5. Set as Public (required for free GitHub Pages)
6. Don't initialize with README

## Step 2: Push to GitHub
Run these commands in your terminal:

```bash
git remote add origin https://github.com/tommyliudev2017-coder/dailyten-website.git
git branch -M main
git push -u origin main
```

## Step 3: Enable GitHub Pages
1. Go to your repository on GitHub
2. Click "Settings" tab
3. Scroll down to "Pages" section
4. Under "Build and deployment", select "Deploy from a branch"
5. Source: "Deploy from a branch"
6. Branch: select "main" and folder "/ (root)"
7. Click "Save"

## Step 4: Get Your Live URL
Your website will be live at:
https://tommyliudev2017-coder.github.io/dailyten-website

## For Google Play Console
Use this URL in your app listing as your website. It will show:
- Your contact email: dailyten@dailytencom.net
- App information
- Professional appearance

This is completely free and never requires a credit card!
